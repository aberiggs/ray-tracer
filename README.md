# ray-tracer
A simple CPU based brute force path tracer written in **C++**. 

<details>
<Summary><h2>Building</h2></Summary>
  
From the root of the repository, do the following:
```sh
mkdir build
cd build
cmake ..
make
```
From this, binaries will be created and stored in `build/bin`.
</details>
